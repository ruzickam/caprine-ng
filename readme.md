#### AppImage:

[Download](https://github.com/ruzickam/caprine-ng/releases/latest) the `.AppImage` file.

Make it [executable](https://discourse.appimage.org/t/how-to-run-an-appimage/80):

```sh
chmod +x caprine_X.XX.X_x86_64.AppImage
```

Then run it!
